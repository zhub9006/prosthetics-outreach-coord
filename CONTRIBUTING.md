# Contributing to Prosthetic Outreach Initiative

## Task Categories
- **Research** — Clinical trial identification, literature reviews
- **Outreach** — Community engagement, event planning
- **Logistics** — Supply chain, shipping coordination
- **Finance** — Fundraising, budget tracking

## Workflow
1. Claim an issue (self-assign)
2. Update status with comments
3. Mark complete when done

## Code of Conduct
Be respectful and inclusive in all communications.
